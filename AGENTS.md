# Agent Instructions

When ending a work session, always:

1. **Update `requirements.bat`** — ensure it installs every dependency the project needs
2. **Update `requirements.sh`** — same for Kali Linux
3. **Update `README.md`** — reflect any new changes, features, or setup steps
4. **`git add -A && git commit -m "..." && git push`** — push everything to GitHub
